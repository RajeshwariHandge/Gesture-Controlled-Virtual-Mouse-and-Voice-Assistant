# 🖱️ Gesture Controlled Virtual Mouse & Voice Assistant

A Python-based virtual mouse and voice assistant that uses hand gestures and voice commands to perform mouse operations without physical contact. The project leverages Computer Vision and Machine Learning techniques for real-time gesture recognition and system control.

## 🚀 Features

- ✋ Cursor Movement using Hand Gestures
- 👆 Left Click
- 👉 Right Click
- ✌️ Double Click
- 📜 Scrolling
- ✊ Drag and Drop
- 🔊 Volume Control
- ☀️ Brightness Control
- 🎤 Voice Assistant Commands
- 🧠 Real-time Hand Tracking using MediaPipe

## 🛠️ Technologies Used

- Python
- OpenCV
- MediaPipe
- PyAutoGUI
- SpeechRecognition
- PyAudio
- pyttsx3
- NumPy

## 🖥️ Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/RajeshwariHandge/Gesture-Controlled-Virtual-Mouse-and-Voice-Assistant.git
cd Gesture-Controlled-Virtual-Mouse-and-Voice-Assistant
```

### 2️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Run the Project

```bash
python Proton.py
```
